# PHP-registration-form

## Start project


1. Open XAMMP htdocs folder
2. Clone repository
``` bash
git clone https://github.com/amosckowsky/php-registration-form.git
```
3. Create .env file and enter your MySQL connection data:
``` bash
    user='{Your user}'
    password='{Your password}'
    dsn='mysql:host={Your host};dbname={Your DB name};charset=utf8mb4'
```
4. Start Apache in XAMMP Control Panel
5. Go to http://localhost/php-registration-form/
